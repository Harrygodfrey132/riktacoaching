      <h1>SNAP-IV 26-Item Parent Rating Scale</h1>
      <p>This questionnaire is a trusted and widely used rating scale to detect possible attention deficit hyperactivity disorder (ADHD). It screens for nine symptoms of ADHD hyperactive-impulsive type, nine symptoms of ADHD inattentive type, and eight symptoms of oppositional defiant disorder. It should be completed by a child's parent or guardian. Symptom severity is rated on a 4-point scale, which you will need to add together once completed.</p>
      <p><strong>For each item, select the box that best describes this child. Put only one check per item.</strong></p>
