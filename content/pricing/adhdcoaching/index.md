<section class="adhd-coaching-breakdown-section" id="how-it-works">
  <div class="container">
    <header class="adhd-coaching-breakdown-section__header">
      <p class="eyebrow">How The Process Works</p>
      <h2>The 6-part coaching framework</h2>
      <p>Our coaching is structured so each session builds on the previous one. We focus on real-life execution, not generic advice.</p>
    </header>
    <div class="adhd-coaching-timeline">
      <article>
        <span>1</span>
        <h3>Mapping and baseline</h3>
        <p>We map current patterns: focus blocks, avoidance triggers, overload points, and where routines break down.</p>
      </article>
      <article>
        <span>2</span>
        <h3>Goal architecture</h3>
        <p>We set 2 to 3 outcomes linked to daily life (school, work, sleep, emotional control, home admin).</p>
      </article>
      <article>
        <span>3</span>
        <h3>Environment design</h3>
        <p>We simplify systems: calendar structure, task capture, reminders, start cues, and transition routines.</p>
      </article>
      <article>
        <span>4</span>
        <h3>Execution practice</h3>
        <p>You test tools in real situations between sessions, then we adapt quickly based on what actually happened.</p>
      </article>
      <article>
        <span>5</span>
        <h3>Regulation and recovery</h3>
        <p>We work on stress signals, emotional reactivity, pacing, and recovery windows to protect consistency.</p>
      </article>
      <article>
        <span>6</span>
        <h3>Maintenance plan</h3>
        <p>When routines stabilize, we shift to lighter follow-up to keep progress without over-reliance on sessions.</p>
      </article>
    </div>
  </div>
</section>
